<h2><b>Instalação</b></h2>

**Baixe a última versão**
```
git clone https://github.com/Alexxmfs/Projetos.git
```

**Mudar para o diretório do projeto**
```
cd lab-ds

# Em seguinda

cd lab-ds
```

**Instalação de dependências**
```
npm install
```

**Exexutar lab-ds**
```
npm run dev
```

**Exexutar StoryBook**
```
npm run storybook
```
